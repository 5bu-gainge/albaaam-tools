한 장 보고서 구조 잡기

흩어진 근거와 메모를 결론 → 핵심 근거 → 요청 순서의 한 장 보고서 골격으로 재배열합니다. 무엇이 결론이고 무엇이 근거인지 짚어 빈칸까지 알려 드립니다.

STEP 1

흩어진 내용 붙여넣기

보고서에 담을 근거·메모·문장을 한 줄에 하나씩 넣으세요. 순서는 뒤죽박죽이어도 됩니다. 다음 단계에서 역할별로 재배열합니다.

입력한 내용은 이 브라우저 밖으로 전송되지 않습니다. 서버·외부 API 호출이 없습니다.

STEP 2

항목 역할 나누기

각 항목의 역할을 자동으로 추정했습니다. 어색한 것은 눌러서 바꾸세요. 결론은 하고 싶은 핵심 메시지, 근거는 그걸 떠받치는 사실, 요청은 읽는 사람에게 바라는 결정입니다.

위에서 내용을 넣고 '항목 나누고 분류하기'를 누르면 여기에 나타납니다.

이 도구는 생각의 뼈대를 잡아 주는 보조 도구이며, 보고서를 대신 완성하거나 내용의 사실 여부를 검증하지 않습니다. 수치·근거는 원본으로 다시 확인하세요.
