핵심가치 포스터 렌더러

이미 확정된 핵심가치명과 한 문장 정의를 넣으면, 정의를 의미 단위로 끊어 (어절을 쪼개지 않는 keep-all 방식) 읽기 좋은 세로 포스터로 배치해 PNG · SVG · 인쇄용으로 저장합니다. 가치 문구 자체는 바꾸지 않습니다.

STEP 1

확정된 핵심가치 입력

각 가치의 이름과 한 문장 정의를 그대로 넣으세요. 영문 라벨은 선택입니다. 정의는 자동으로 의미 단위 줄바꿈이 적용됩니다.

STEP 2

포스터 옵션

선택된 파일 없음

투명 배경 PNG 권장. 우측 하단에 배치됩니다. 브랜드명보다 로고가 우선합니다.

입력한 내용과 이미지는 이 브라우저 안에서만 처리되며, 외부로 전송되지 않습니다.

STEP 3

포스터 결과

▤

아직 생성된 포스터가 없습니다.

가치를 입력하고 '포스터 생성'을 누르세요. '예시로 채우기'로 먼저 체험할 수 있습니다.
